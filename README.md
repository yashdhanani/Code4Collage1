## 🛠️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/yashdhanani/Code4Collage.git
cd Code4Collage
```

### 2. Install Dependencies

#### Frontend
```bash
cd frontend
npm install
npm run dev
```

#### Backend
```bash
cd backend
npm install
npm run dev
```

### 3. Setup `.env`

Create a `.env` file and add:

```env
PORT=5000
MONGO_URI=your-mongo-uri
JWT_SECRET=your-secret-key
```
